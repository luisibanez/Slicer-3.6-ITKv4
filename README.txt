
This repository contains a copy of the release branch
of Slicer version 3.6.

It is intended to be built againts a development version
of ITKv4. Compilation errors from this build will be used
to feed an ITKv4 migration guide database.

Details about the ITKv4 migration plans are available at:

  http://www.itk.org/Wiki/ITK_Release_4/Migration_Plan

----------------------------------------------------------

For more information, please see:

  http://www.slicer.org

and

  http://wiki.na-mic.org/Wiki/index.php/Slicer3
